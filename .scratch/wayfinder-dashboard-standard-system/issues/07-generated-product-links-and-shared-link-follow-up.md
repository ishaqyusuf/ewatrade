# 07 - Generated Product Links And Shared-Link Follow-Up

**What to build:** owners and permitted staff can manage generated product links, inspect link activity, review shared-link order requests, and complete or cancel follow-up from the dashboard.

**Blocked by:** 03 - Products Proof Slice With Table, Sheet, And Form Foundations; 06 - Sales, Sessions, Customers, And Closeout Review.

**Status:** ready-for-agent

- [ ] Generated product links can be listed with product, creator, status, views, orders, and last activity where available.
- [ ] Users can create and deactivate product links through permissioned dashboard actions.
- [ ] Shared-link order requests are visible with customer, selected product/unit, payment, pickup or delivery, reservation, and notification state where available.
- [ ] Permitted users can complete or cancel shared-link order requests and see resulting status changes.
- [ ] Delivery request state is surfaced for shared-link orders where existing delivery APIs provide it.
- [ ] Browser QA covers link list, link creation, deactivation, shared-link order review, complete/cancel follow-up, delivery status visibility, and permission boundaries.
- [ ] API or Brain docs are updated if dashboard link or follow-up behavior changes.
