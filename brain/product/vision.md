# Product Vision

## Purpose
Capture the product direction and who the platform is for.

## How To Use
- Update when target market, positioning, or core outcomes change.

## Vision
Build commerce and logistics infrastructure for African merchants that combines storefronts, merchant operations, fulfillment coordination, and communication workflows in one multi-tenant platform — anchored by a main website that shows the ecosystem alive and growing.

## Primary Users
- Merchants (store owners)
- Logistics / dispatch providers
- Drivers
- Store cashiers
- End customers

## Differentiation
- Commerce plus logistics in one platform
- Every tenant (store or logistics) gets their own generated website
- Main platform website is a hybrid advertisement + storefront discovery + live activity feed — not a conventional ecommerce site
- Strong multi-store and multi-tenant model
- Shared customer identity across all tenant sites (one login for all EwaTrade stores)
- Website builder and marketplace support
- Messaging and assisted checkout workflows

## Main Site as a Living Showcase
The main EwaTrade website serves as proof of a thriving ecosystem:
- Visitors can see real-time platform activity (orders, dispatches, registrations, product engagement)
- Featured merchants, logistics companies, and drivers are advertised on the front page
- Store products with promotions or subscriptions are discoverable
- The only call-to-action is to register — merchants, logistics providers, and drivers onboard from here
- Customers use one credential set to shop across any tenant site
