
# Marketplace

Optional product discovery marketplace.

Goals:

- help merchants get traffic
- promote merchant brand
- increase product exposure

Rules:

- merchant name always visible
- merchant website link visible

## Product Image Marketplace Wayfinder

The product-image marketplace Wayfinder is resolved in `.scratch/wayfinder-product-image-marketplace/`.

Resolved direction:

- Merchants explicitly opt images into an in-platform marketplace license.
- Buyers spend tokens for copied in-platform asset rights, not ownership transfer.
- Tenant-owned token balances use ledger entries for top-ups, holds, debits, earned credits, refunds, disputes, and payout reversals.
- Native mobile token purchases should use Apple/Google billing paths for digital image tokens; web/dashboard top-ups can use normal provider checkout.
- Marketplace search can show public listings and the merchant's own gallery only.
- Moderation, dispute, refund, and lineage audit are required before launch.

See `.brain/features/product-image-marketplace-and-storefront-publishing.md`.
