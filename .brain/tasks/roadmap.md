# Roadmap

## Purpose
Longer-horizon sequencing across major implementation phases.

## Phases
1. Foundation: repo scaffolding, auth, tenancy, database schema, shared packages.
2. Commerce Core: stores, products, inventory, carts, orders, payments integration.
3. Logistics: dispatch onboarding, bids, assignments, tracking.
4. Channels: storefront builder, marketplace, WhatsApp commerce.
5. Retail Ops: POS, self-service checkout, analytics, automation.
